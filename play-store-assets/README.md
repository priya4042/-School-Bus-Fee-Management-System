## 📖 Complete Guide

### STEP 1: Create App Icon

**Read**: Page 1 in `UPLOAD_GUIDE.md`  
**Time**: 5-15 minutes  
**Tool Options**: 
- Android Asset Studio (easiest)
- Figma (most control)
- Canva (templates)

**Requirements**:
- Size: **512×512 pixels** (exactly)
- Format: PNG (recommended)
- No transparency
- Professional design
- School bus theme recommended

**Save to**: `app-icon/app_icon_512x512.png`

---

### STEP 2: Create Feature Graphic

**Read**: Page 2 in `UPLOAD_GUIDE.md`  
**Time**: 10-15 minutes  
**Tool Options**:
- Figma (best results)
- Canva (easy templates)

**Requirements**:
- Size: **1024×500 pixels** (exactly)
- Format: PNG or JPEG
- Landscape orientation
- Include text: "BusWay Pro", key features
- Colors: Blue (#1e40af) background

**Save to**: `feature-graphic/feature_graphic_1024x500.png`

---

### STEP 3: Create Screenshots (5 files)

**Read**: Page 3 in `UPLOAD_GUIDE.md`  
**Time**: 30-45 minutes  
**Tool Options**:
- Real Android phone (best, most authentic)
- Android Emulator (free, easy)
- Figma mockups (alternative)

**Requirements**:
- Size: **1242×2208 pixels** (exactly, each)
- Format: PNG or JPEG
- Vertical orientation (tall, narrow)
- 5 different screens
- No personal data
- Large, readable text

**Screens to Capture**:
1. Dashboard/Home screen
2. Fee Payment screen
3. Bus Tracking (map)
4. Attendance history
5. Settings/Menu

**Save to**: `screenshots/screenshot_1_*.png` through `screenshot_5_*.png`

---

### STEP 4: Upload to Play Console

**Read**: `guides/PLAY_STORE_URLS.md`  
**Time**: 10 minutes  
**Files to upload**: All created images

**URLs**:
- App icon: https://play.google.com/console → Setup → App icon
- Feature graphic: https://play.google.com/console → Setup → Graphics assets
- Screenshots: https://play.google.com/console → Setup → Graphics assets → Phone screenshots

**Steps**:
1. Go to Play Console
2. Select your app
3. Upload each image to correct location
4. Click Save
5. Verify in preview

## 🎨 Design Tips

### Colors to Use
```
Primary Blue:         #1e40af
Dark Blue:            #1e3a8a
School Bus Yellow:    #fbbf24
White:                #ffffff
Dark Gray:            #1f2937
Light Gray:           #f3f4f6
```

### What Makes Good Screenshots
✅ Show real app features  
✅ Use bright, attractive colors  
✅ Include descriptive text  
✅ Show different screens in order  
✅ No personal information  
✅ No test accounts visible  
✅ Text large and readable  
✅ Professional appearance  

❌ Avoid blurry images  
❌ Avoid very small text  
❌ Avoid private data (emails, names)  
❌ Avoid wrong orientation  
❌ Avoid inconsistent styling
