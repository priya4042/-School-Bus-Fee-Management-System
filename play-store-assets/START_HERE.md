## 📐 SIZE SPECIFICATIONS

Bookmark this for reference:

```
APP ICON
├─ Size: 512×512 pixels (EXACTLY)
├─ Format: PNG (recommended) or JPEG
├─ Transparency: NO (solid background required)
├─ File size: < 512 KB
└─ Quality: Professional, clear, recognizable

FEATURE GRAPHIC
├─ Size: 1024×500 pixels (EXACTLY)
├─ Format: PNG or JPEG
├─ Orientation: Landscape (wide, not tall)
├─ Aspect ratio: 2.048:1
├─ File size: < 5 MB
└─ Quality: High, with text

PHONE SCREENSHOTS (×5-8)
├─ Size: 1242×2208 pixels (EXACTLY - each)
├─ Format: PNG or JPEG
├─ Orientation: Portrait (tall, narrow)
├─ Aspect ratio: 9:16
├─ Quantity: Minimum 2, recommended 5-8
├─ File size: < 5 MB each
└─ Quality: High, no personal data
```

## 🚨 IF YOU GET STUCK

### Common Questions

**Q: Can't find Android Asset Studio URL?**  
A: It's at: https://romannurik.github.io/AndroidAssetStudio/

**Q: How do I check image size?**  
A: Use: https://www.imageresizer.com/ or right-click → Properties

**Q: Can I use JPEG instead of PNG?**  
A: Yes! Both work. PNG is better for quality.

**Q: My image is 513×513, is that OK?**  
A: NO. Must be EXACTLY 512×512. Resize it to exact size.

**Q: How do I resize an image?**  
A: Use: https://www.imageresizer.com/ or Figma or Canva

**Q: My file is too large (>5MB)?**  
A: Compress at: https://tinypng.com/

**Q: Can't upload to Play Console?**  
A: Check in PLAY_STORE_URLS.md "Troubleshooting" section
