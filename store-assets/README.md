# Store Assets Folder

This folder contains ready-to-upload listing assets for Google Play and Apple App Store.

## Files

- `playstore-icon-512.png` (Google Play app icon)
- `playstore-feature-graphic-1024x500.png` (Google Play feature graphic)
- `appstore-icon-1024.png` (Apple App Store icon)
- `brand-logo-1800x560.png` (marketing/logo usage)

## Regenerate assets

If you update branding files in `resources/branding`, run:

```bash
npm run assets:generate
```
