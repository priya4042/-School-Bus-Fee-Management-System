```
You are a Google Play Store listing expert. I need help creating professional store listing content for my Android app.

PROJECT DETAILS:
================
App Name: BusWay Pro
Package Name: com.buswaypro.app
Target Users: Parents, School Administrators
App Category: Education / Transportation

WHAT THE APP DOES:
==================
BusWay Pro is a comprehensive school bus management system that helps:
- Parents track their child's bus location in real-time using GPS
- View bus attendance (pickup & drop-off times)
- Pay school bus fees online (Razorpay & Stripe integration)
- Receive notifications about bus status and payment reminders
- View receipts and payment history
- Access student profile and enrollment information
- Manage multiple students under one parent account

KEY FEATURES:
=============
1. Real-time Bus Location Tracking - Live GPS tracking of buses on a map
2. Attendance Management - See pickup and drop-off records
3. Online Fee Payments - Pay through Razorpay or Stripe securely
4. Instant Notifications - Push/SMS alerts for important updates
5. Digital Receipts - Automatic receipt generation for payments
6. Multi-Student Support - Manage multiple children in one app
7. Parent Dashboard - Overview of fees, attendance, and notifications
8. Admin Portal - For school staff to manage routes, buses, and payments
9. WhatsApp Integration - Option to receive notifications via WhatsApp
10. Security Features - Secure login with OTP verification

TECH STACK:
===========
Frontend: React 19, TypeScript, Vite
Mobile: Capacitor (Android)
Backend: Node.js
Database: PostgreSQL (Supabase)
Payments: Razorpay, Stripe
SMS/OTP: Twilio
Maps: Leaflet + OpenStreetMap (completely free)
Deployment: Vercel (frontend), Render (backend)

PRIMARY BENEFITS FOR PARENTS:
=============================
- Know exactly where the bus is, in real-time
- Never miss an attendance update
- Easy online payment with multiple options
- Transparent fee tracking and receipts
- Get alerts about fees and updates
- One app to manage multiple children

PRIMARY BENEFITS FOR SCHOOLS/ADMINS:
====================================
- Manage multiple bus routes efficiently
- Track student attendance automatically
- Collect fees online (reduce manual collection)
- Send bulk notifications to parents
- Generate payment and attendance reports
- Reduce administrative work

WHAT I NEED HELP WITH:
======================
Please guide me through creating these EXACT items in order:

1. APP TITLE (30 characters max)
   - Short, catchy name for Play Store listing

2. SHORT DESCRIPTION (50-80 characters)
   - One-liner hook that appears in search results

3. FULL DESCRIPTION (4000 characters max)
   - Detailed description of what app does
   - Benefits for parents and schools
   - Key features listed
   - Call to action

4. PRIVACY POLICY
   - Give me a template/structure for privacy policy
   - What should I mention (data collection, security, payments)

5. SUPPORT EMAIL
   - Template for support contact info

6. CONTENT RATING QUESTIONNAIRE
   - Guide me on how to answer Play Store content rating questions

7. DATA SAFETY
   - What to declare (permissions used, data handling)

8. SCREENSHOTS DESCRIPTIONS (2-4 phone screenshots)
   - For each screenshot, tell me what it should show and caption text
   - Example: "Screenshot 1: Show the map with bus location, caption: 'Track Your Bus in Real-Time'"

9. FEATURE GRAPHIC (1024×500 px)
   - Describe what this image should contain
   - Text styling suggestions
   - Color scheme that matches the app

10. RELEASE NOTES (FOR FIRST VERSION)
    - Template for initial release notes

FOR EACH SECTION:
=================
- Give me the EXACT text I should use (ready to copy-paste)
- Explain WHY this text works for Play Store
- Give me 2-3 alternative versions if applicable
- Explain what NOT to do (common mistakes)

IMPORTANT RULES:
================
- Keep language professional but friendly
- Focus on benefits, not just features
- Make it clear who the app is for
- Use simple, clear language
- Don't use marketing hype or false claims
- Mention security and privacy prominently
- Include a call to action at the end

STEP-BY-STEP PROCESS:
====================
1. Start with APP TITLE - wait for my confirmation
2. Then SHORT DESCRIPTION - wait for my confirmation
3. Then FULL DESCRIPTION - wait for my confirmation
4. Continue with remaining sections one by one
5. After each section, I'll tell you if I want changes
6. Don't rush - give me time to review each section

Please start with the APP TITLE and give me 3 options with explanations.
```
