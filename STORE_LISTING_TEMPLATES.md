# Store Listing Templates (BusWay Pro)

Use this file to quickly fill both Google Play Console and App Store Connect.

## 1) App identity

- App name: BusWay Pro
- Subtitle / short branding: School Bus Fee Management
- Package ID (Android): `com.buswaypro.app`
- Bundle ID (iOS): `com.buswaypro.app` (set this in Xcode when adding iOS)

## 2) Google Play listing text

### Short description (80 chars max)

Smart school transport app for bus tracking, fee payments, and parent alerts.

### Full description

BusWay Pro helps schools and parents manage daily school transport operations with confidence.

Key features:
- Real-time bus tracking and parent visibility
- Secure fee collection and payment records
- Digital receipts and payment history
- Route and bus assignment management
- Parent notifications for trip and fee updates
- Admin dashboards for reporting and operations

BusWay Pro is designed for schools that need reliable transport operations, transparent fee workflows, and a modern parent experience.

## 3) Apple App Store listing text

### Subtitle (30 chars max)

School Bus Fee Management

### Promotional text

Track buses live, collect fees securely, and keep parents informed in real time.

### Description

BusWay Pro is a complete school transport operations platform for administrators and parents.

What you can do:
- Monitor live bus movement for safer student transport
- Manage monthly transport fees and payment workflows
- Generate and access digital receipts
- Send timely updates and alerts to parents
- Review reports for collections and operations

Built for schools that need dependable transport, billing transparency, and better communication.

## 8) Common rejection prevention checks

- App icon and screenshots must match actual UI branding
- Privacy policy must mention location/camera/payment usage
- Broken links in listing metadata should be fixed before submit
- Ensure app does not crash on first launch with production config
- Use production API endpoints and HTTPS only
